# Inventory Optimization & Vendor Analytics

## Overview

Developed an end-to-end retail analytics solution using PostgreSQL and Power BI to optimize inventory management, evaluate vendor performance, and analyze sales trends through interactive dashboards.

## Business Problem

Retail businesses often struggle with:

* Excess inventory and stock holding costs
* Limited visibility into vendor performance
* Inefficient product inventory allocation
* Lack of centralized business reporting

This project helps stakeholders monitor inventory, evaluate suppliers, and identify high-performing products using data-driven insights.
